#  AI & Machine Learning — Complete Study Guide

> A comprehensive, bilingual (EN/FR) interactive study guide for AI & Machine Learning.

🔗 **Live:** [fstt-learn-ia.netlify.app](https://fstt-learn-ia.netlify.app/)

## Overview

Single-page HTML reference covering the full ML pipeline — from Python fundamentals (NumPy, Pandas) to training, evaluating, and deploying models with scikit-learn.

### What's Inside

- **NumPy & Pandas** essentials with code examples
- **7-step ML pipeline** (Collect → EDA → Clean → Feature Eng. → Train → Evaluate → Tune)
- **Algorithm guides** — Logistic Regression, Random Forest, SVM, KNN, Decision Trees, K-Means, PCA
- **Full example project** using urban density classification data
- **Library cheatsheet** & common mistakes table
- 🇬🇧/🇫🇷 Language toggle with localStorage persistence

## Tech

Static HTML/CSS/JS — no build tools, no dependencies.
